En esta carpeta encontrarás 4 archivos importantes para poder ejecutar el programa de Encantia. Cada uno corresponde a un sistema operativo diferente:

1. Archivo .exe (Windows)
Este archivo se utiliza para ejecutar el programa en computadoras con sistema operativo Windows.

2. Archivo .deb (Linux)
Este archivo permite instalar y ejecutar el programa en sistemas operativos Linux.

3. Archivo .app (MacOS)
Este archivo está diseñado para ejecutar el programa en computadoras con MacOS.

4. Archivo .vsix
Este archivo es una extensión (plugin) para el editor de código.

    Sirve para:
        - Ayudarte a escribir correctamente el código
        - Detectar errores mientras programas
        - Guiarte en el uso de la sintaxis del lenguaje Encantia

Al ejecutar el archivo correspondiente a tu sistema operativo, se abrirá una ventana donde podrás comenzar a programar en el lenguaje Encantia.

Importante:
    Antes de ejecutar el programa, asegúrate de leer el archivo README general, que se encuentra hasta abajo de las carpetas.
    Ahí se explican los pasos necesarios para ejecutar correctamente el programa.
