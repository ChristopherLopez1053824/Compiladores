# Encantia Lang

Extensión de VS Code para el lenguaje Encantia.

## Características

- Resaltado de sintaxis para archivos `.encantia`
- Botón para ejecutar programas Encantia
- Ejecución mediante compilador incluido en la extensión

## Requisitos

Se necesita tener instalado Java JDK para ejecutar programas Encantia.
