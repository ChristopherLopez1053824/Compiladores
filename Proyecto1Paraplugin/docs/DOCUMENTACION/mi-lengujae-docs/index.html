📖 Sobre Encantia

Este documento presenta la especificación completa de nuestro lenguaje de programación. Encantia no es solo código… es una historia.

Inspirado en un mundo de fantasía, este lenguaje fue diseñado como si fuera un cuento, donde cada instrucción es un hechizo y cada programa una historia que cobra vida.

A través de esta documentación, se describen sus características fundamentales y las palabras reservadas que forman este universo mágico.

🌟 Filosofía del Lenguaje

Encantia es distinto al resto, ya que no está orientado a objetos. De hecho, se acerca más a un pseudocódigo, permitiendo que cualquier persona pueda entender su funcionamiento sin complejidad innecesaria.

El uso de clases, objetos y herencia fue omitido intencionalmente para mantener un lenguaje claro, directo y poderoso en su simplicidad.

El objetivo es que el usuario domine completamente:

  • ✨ Variables
  • ✨ Expresiones
  • ✨ Condicionales
  • ✨ Ciclos
  • ✨ Funciones simples

🧠 Aprendizaje Práctico

Para facilitar el entendimiento, Encantia presenta ejemplos prácticos que permiten al usuario visualizar cómo funciona cada parte del lenguaje en escenarios reales.

Aquí no solo lees… aquí practicas, experimentas y aprendes haciendo.
