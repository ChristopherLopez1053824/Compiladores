🔁 Ciclos

Los ciclos en Encantia permiten repetir un bloque de código varias veces sin necesidad de escribirlo repetidamente. Esto es útil cuando se desea ejecutar una acción muchas veces, como mostrar mensajes, recorrer datos o realizar cálculos.

Encantia cuenta con dos tipos principales de ciclos:

  • viaje → se usa cuando sabes cuántas veces quieres repetir algo (equivalente a for)
  • guardian → se usa cuando la repetición depende de una condición (equivalente a while)

✨ Ejemplo 1: Uso de viaje

viaje (intenso i = 0; i < 3; i++) {
    Divulga("Número: " + i);
}

🔍 Explicación:

  • intenso i = 0; → se crea una variable que inicia en 0
  • i < 3 → el ciclo se ejecuta mientras i sea menor que 3
  • i++ → en cada repetición, i aumenta en 1
  • Divulga(...) → imprime el valor actual de i

🔁 Resultado:

Número: 0
Número: 1
Número: 2

✨ Ejemplo 2: Uso de guardian

intenso i = 0;

guardian (i < 3) {
    Divulga("Valor: " + i);
    i = i + 1;
}

🔍 Explicación:

  • Primero se crea la variable i
  • El ciclo se ejecuta mientras i < 3
  • Dentro del bloque:
  • Se imprime el valor
  • Se incrementa manualmente (i = i + 1)
  • Si no se incrementa, el ciclo nunca terminaría ⚠️
