🧩 Condicionales

Los condicionales permiten que el programa tome decisiones dependiendo de si una condición es verdadera o falsa. Esto hace que el programa pueda comportarse de diferentes maneras según los datos que reciba.

Encantia usa:

  • destino → si se cumple la condición (if)
  • alterno → si no se cumple (else)

✨ Ejemplo 1

intenso edad = 20;

destino (edad >= 18) {
    Divulga("Eres mayor de edad");
}

🔍 Explicación:

  • Se declara una variable edad
  • Se evalúa si edad >= 18
  • Como 20 es mayor que 18, se ejecuta el bloque
  • Si no se cumpliera, no pasaría nada (porque no hay alterno)

✨ Ejemplo 2

intenso edad = 15;

destino (edad >= 18) {
    Divulga("Puedes entrar");
} alterno {
    Divulga("Acceso denegado");
}

🔍 Explicación:

  • Se evalúa la condición
  • Como 15 no es mayor o igual a 18:
  • Se ignora el bloque de destino
  • Se ejecuta alterno
  • Esto permite manejar dos posibles caminos
