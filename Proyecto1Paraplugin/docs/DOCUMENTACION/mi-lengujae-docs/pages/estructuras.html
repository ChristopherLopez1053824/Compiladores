✨ Estructuras Especiales

Estas estructuras controlan el inicio del programa y la forma en que se muestra información al usuario.

  • comienza → punto de inicio del programa (main)
  • Divulga → imprime información en pantalla

✨ Ejemplo 1

comienza {
    Divulga("Bienvenido a Encantia");
}

🔍 Explicación:

  • comienza indica dónde inicia la ejecución
  • Todo lo dentro del bloque se ejecuta al iniciar
  • Divulga muestra el mensaje en pantalla

✨ Ejemplo 2

comienza {
    Cancion nombre = "Nicolle";
    Divulga("Hola " + nombre);
}

🔍 Explicación:

  • Se crea una variable tipo texto
  • Se concatena con un mensaje
  • Se imprime el resultado
