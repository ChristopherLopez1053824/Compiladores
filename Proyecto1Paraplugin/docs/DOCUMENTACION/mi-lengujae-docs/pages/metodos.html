⚙️ Métodos

Los métodos, llamados hechizos en Encantia, son bloques de código que realizan una tarea específica. Sirven para reutilizar código y organizar mejor el programa.

Se definen usando la palabra hechizo, indicando:

  • el tipo de retorno
  • el nombre
  • los parámetros (opcional)

✨ Ejemplo 1: Hechizo sin retorno

hechizo vasto saludar() {
    Divulga("Hola viajero");
}

🔍 Explicación:

  • vasto significa que no devuelve ningún valor
  • saludar() es el nombre del método
  • Dentro del bloque se ejecuta un print
  • Este método solo realiza una acción

✨ Ejemplo 2: Hechizo con retorno

hechizo intenso sumar(intenso a, intenso b) {
    felices a + b;
}

intenso resultado = sumar(3, 4);
Divulga(resultado);

🔍 Explicación:

  • El método recibe dos valores (a y b)
  • felices devuelve el resultado de la suma
  • El valor retornado se guarda en resultado
  • Luego se imprime
