🧙‍♂️ Variables

Las variables son espacios en memoria donde se almacenan datos. En Encantia, cada variable debe tener un tipo que define qué tipo de dato puede guardar.

Tipos disponibles:

  • intenso → números enteros
  • dorado → números decimales
  • fenix → números flotantes
  • chispa → caracteres
  • Cancion → texto
  • bondad → verdadero o falso
  • vasto → sin valor (solo para métodos)

✨ Ejemplo 1

intenso numero = 10;
dorado precio = 25.5;
Cancion nombre = "Encantia";

🔍 Explicación:

  • Se crean tres variables con distintos tipos
  • Cada una guarda un valor específico
  • El tipo determina qué tipo de datos puede almacenar

✨ Ejemplo 2

bondad activo = true;
chispa letra = 'A';

Divulga(activo);
Divulga(letra);

🔍 Explicación:

  • bondad guarda valores booleanos (true/false)
  • chispa guarda un solo carácter
  • Se imprimen los valores usando Divulga
