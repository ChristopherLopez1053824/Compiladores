# Aqui se escrbira que debemos de hacer donde ver la documentacion y donde etan los ejecutables

1. Instalar Java

Antes de todo, es necesario tener Java instalado en tu computadora.
Esto se debe a que Encantia está basado en este lenguaje.

Puedes descargarlo desde la página oficial en el siguiente enlace:
https://www.java.com/es/download/manual.jsp

Si no tienes Java instalado, el programa de Encantia no funcionará correctamente.


2. Revisar la documentación

Dentro del proyecto encontrarás una carpeta llamada “docs” y es importante leer este documento antes de empezar a programar.
Ahí hay un archivo en formato PDF donde se explica:

    - Las palabras reservadas del lenguaje
    - La sintaxis que debes utilizar
    - Ejemplos básicos

O pueden ingresar al siguiente link, donde se encuentra de forma más dinámica y expresiva la forma de programar en el lenguaje encantia: https://encantia.doppio.cloud/


3. Usar los ejecutables 

En la carpeta “Ejecutables” encontrarás dos archivos importantes:

    • Archivo .exe (Este archivo no permite la funconalidad convoca)
        Este archivo te permite ejecutar el programa de Encantia.  

        Al abrirlo, verás una ventana donde debes seguir este orden de pasos:
            1. Abrir (si ya tienes un programa guardado en un archivo .txt)
            2. Analizar
            3. Ejecutar
            4. Limpiar o Guardar

    • Archivo .vsix (Este archivo permite la funconalidad convoca)
        Este archivo es una extensión (plugin) para el editor de código.

        • Para instalar el plugin existen dos formas posibles de realizarlo:

        1. La primera es descargandolo desde las extensiones de Visual Studio
            - Ese apartado se encuentra del lado izquierdo y es la quinta figura de arriba hacia abajo.
            - En el buscador pegan el nombre del plugin: encantia-lang
            - Seleccionan el que tiene una figura de sombrero mágico y le presionan instalar.

        2. La segunda forma es usando el archivo .vsix
            - Se dirigen a la carpeta de "Ejecutables" y seleccionan el archivo "encantia-0.0.2.vsix" con el click derecho.
            - Luego seleccionan la opción que diga "Instalar la extension VSIX".
            - Posteriormente, se te mostrará una pequeña notificacion en la esquina inferior derecha, mostrando que se instalo    correctamentela extensión.
