📦 Importaciones

Las importaciones permiten incluir funcionalidades externas dentro del programa. Esto es útil para usar herramientas ya existentes sin tener que programarlas desde cero.

Se utiliza la palabra convoca seguida del nombre del recurso, que debe estar dentro de comillas dobles. Tomando en cuenta que debe ser un archivo existente en la misma carpeta.

✨ Ejemplo 1

convoca "magia.encantia";

🔍 Explicación:

  • Se importa un paquete llamado magia.utilidades
  • Permite usar funciones o herramientas definidas ahí

✨ Ejemplo 2

convoca "sistema.encantia";

🔍 Explicación:

  • Se importa un módulo que podría servir para entrada de datos
  • Esto permite usar funciones como escuchar
